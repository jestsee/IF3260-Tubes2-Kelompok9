Help Menu

  1. On the left side, there is a canvas displaying the object model.
  2. To change the angle of the object display, you can use the slider located on the right.
  3. To zoom in or out, you can use the + and - buttons located below the angle slider.
  4. To reset the object view to the default display, you can use the 'reset view' button.
  5. To change the projection, you can use the projection drop-down menu.
  6. For per-object modifications such as FOV, rotation, translation, and scale, you can use the available sliders. However, ensure that the desired object is selected in the 'select an object' drop-down menu.
  7. To change the angle of the oblique projection, you can use the slider inside the box.
  8. To save and load object files, you can use the save and load buttons located at the top.
